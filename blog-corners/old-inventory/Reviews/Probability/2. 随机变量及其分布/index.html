# 2.2.2 伯努利试验（二项分布）

设有一个 EEE，只有两个结果，现将 EEE 进行 n 次独立的重复试验，称为 n 重伯努利试验。

伯努利试验满足二项分布：

P{X=k}=Cnkpk(1−p)n−k,k=0,1,2…nP\{X = k\} = C_n^k p^k (1 - p)^{n - k} \ , k = 0,1,2 \ldots n P{X=k}=Cnk​pk(1−p)n−k,k=0,1,2…n

简化为：B(n,p)B(n, p)B(n,p)
